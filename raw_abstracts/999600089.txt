Blood Bowl is a fantasy football game created by Jervis Johnson for the British games company Games Workshop as a parody of American Football.<ref name="HG"></ref> The game was first released in 1986 and has been re-released in new editions since. Blood Bowl is set in an alternate version of the Warhammer Fantasy setting, populated by traditional fantasy elements such as human warriors, goblins, dwarves, elves, orcs and trolls.
In late 2016, Games Workshop released a new version of the game - the first in 22 years.<ref></ref> It featured a double sided board and new plastic miniatures.<ref name =Fest16></ref>
In August 2020, Games Workshop announced the latest version of the game, titled Blood Bowl Second Season Edition, which features a significant overhaul of the rules.<ref></ref> This version was released in November 2020 and included miniatures for two teams and referees, a board (pitch), templates and the rule book. The rule book was also available separately, both physically and digitally. Cyanide Studio confirmed that the next videogame adaptation, Blood Bowl 3, would use the new ruleset.<ref></ref>
